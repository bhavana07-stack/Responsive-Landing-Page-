# EcoRide Landing Page

## Project Overview

EcoRide is a responsive landing page for an eco-friendly electric bike rental startup. The project is designed to provide a modern and user-friendly interface for promoting sustainable transportation.

## Features

* Responsive design for desktop and mobile devices
* Modern hero section with animations
* Interactive information buttons
* Attractive green-themed user interface
* Smooth hover effects and transitions

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

## How to Run

1. Download the project files.
2. Open `index.html` in any web browser.
3. Explore the landing page.
